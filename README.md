# House Price Prediction Web App

## Overview
This project is a Machine Learning web application that predicts median house prices based on housing features using the California Housing Dataset.  
The trained regression model is deployed using Flask and provides real-time predictions through a simple web interface.

---

## Features
- House price prediction using Machine Learning regression models
- Data preprocessing and feature scaling
- Flask-based backend deployment
- User-friendly web interface
- Real-time prediction output

---

## Tech Stack
- Python
- Scikit-Learn
- Flask
- Pandas & NumPy
- HTML, CSS

---

## Project Workflow
1. Data preprocessing and exploratory data analysis (EDA)
2. Model training using regression algorithms
3. Model evaluation using R² score and Mean Squared Error (MSE)
4. Model serialization using pickle
5. Deployment using Flask web application

---

## Project Structure
app.py – Flask application
model.py – model loading/training logic
house_model.pkl – trained ML model
templates/ – HTML files
static/ – CSS styling
